# プライベートCFO — コンセプト

## サービス概要

高所得者向けの月額制・招待制「支出管理代行」サービス。
顧客は自身の支出を入力/確認し、本部(CFO チーム)が
代行入力・レポーティング・アドバイスを行う。

## 顧客体験の核

- 「記帳を自分でやらなくていい」安心感
- 本部からのテロップ・PDF・ポイントで "見守られている" 感
- ネイビー×ゴールドの落ち着いた UI

## 役割

本アプリは **顧客(client)専用**。
本部(admin)は別アプリ(`Desktop/admin175-project`)から操作する。
両アプリは同じ Supabase プロジェクトを共有し、RLS で権限を分離する。

- **client** : 自分の支出を閲覧・入力、本部への連絡が可能
- **admin**  : 全顧客を横断、代行入力、テロップ/PDF/ポイント送信(本部アプリで完結)

## スコープ外(明記)

- 決済処理(銀行振込・クレジットカード、アプリ外)
- 税務申告書の生成
- 顧客同士のやり取り
- 顧客アプリからの招待発行(招待は本部アプリ側のみ)

  Supabase 認証 + DB に置き換える形で本番化する
- カラー・アイコン・カテゴリ・フォーマッタは
  `Desktop/shared-cfo` から本部アプリと共有
- 顧客データは Supabase RLS で他顧客から隔離、
  admin ロールのみ横断参照可
